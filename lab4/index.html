This is an almost minimal command-based live coding engine. Press the button to re-evaluate your code. This language allows you to specify a rhythmic pattern of beats by entering a space-separated list of times, pitches, and osc types (sine, sawtooth, triangle, or square). You can also include javascript computation in the values. Making a time or pitch value negative will make it a random value sampled from [0, |value|]. As an example:

1+1@220@sawtooth 2@-220*2 -3@270@triangle
